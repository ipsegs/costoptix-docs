## Coming Soon

Cost Optix will support Kubernetes cost visibility via a lightweight **in-cluster agent** that collects namespace, pod, and node resource metrics and maps them to cost estimates — with no billing lag.

Planned capabilities include:

- Per-namespace and per-workload cost breakdown
- CPU and memory request vs. actual usage
- Node pool cost attribution
- Works alongside your existing cloud provider accounts (AKS, EKS, GKE) for correlated cloud + cluster costs
