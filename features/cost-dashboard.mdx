## Overview

The Cost Dashboard aggregates spend from every connected cloud provider into a single filterable view. Instead of switching between AWS Cost Explorer, Azure Cost Management, and GCP Billing separately, you see everything in one place.

## Filtering

Use the account switcher in the top navigation to scope the dashboard to:

- **All accounts** — aggregated spend across every connected provider
- **A specific account** — drill into a single AWS account, Azure subscription, GCP project, or Heroku account

The date range picker lets you select preset ranges (last 7 days, last 30 days, this month, last month) or a custom date range.

## Cost Breakdown Views

### Service-level

The default view shows spend grouped by cloud service (e.g. EC2, Azure Virtual Machines, BigQuery). This maps directly to how providers categorise costs on their invoices.

### Resource-level

Drill below service totals to see individual resources — specific VMs, storage buckets, functions, and databases — and exactly how much each one costs. Resource-level data requires providers that support it (AWS and Azure support this natively; GCP requires the Detailed usage cost BigQuery export).

### Provider breakdown

When multiple providers are connected, the dashboard shows a provider split so you can see the proportion of spend coming from AWS vs Azure vs GCP at a glance.

## Historical Trends

Cost Optix stores historical cost data up to your plan's retention limit:

| Plan | Historical Data |
|---|---|
| Starter | 30 days |
| Professional | 90 days |
| Business | 12 months |
| Enterprise | Unlimited |

Trend lines show period-over-period comparisons so you can immediately see whether costs are growing, stable, or declining.

## Portfolio Analytics

The dashboard includes portfolio-level analytics across all accounts:

- **Total spend** for the selected period
- **Month-to-date vs previous month** comparison
- **Largest cost drivers** by service and by account
- **Budget risk summary** — how many budgets are at warning or critical status

## Exporting Data

Use the **Export** button on any cost view to download a CSV of the currently visible data. Export rate limits apply per your subscription tier (up to 300 exports per hour on paid plans).
